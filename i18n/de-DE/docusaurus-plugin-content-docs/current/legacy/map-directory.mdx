Welcome to the complete list of Subway Builder maps! This list has (almost) every custom map made
for Subway Builder. If you know of a map that is not on this list or see any issues, please
[open an issue](https://github.com/Subway-Builder-Modded/Subway-Builder-Modded-Docs/issues) and I
will look at it as soon as time allows. Thank you!

## Installation {#installation}

The installation process is different depending on how and when the map was made by the modder. For
maps that use
[Kronifer's Map Manager](https://github.com/Subway-Builder-Modded/subwaybuilder-patcher/releases),
they all use the same, streamlined process, which can be found at the
[installation guide](/wiki/legacy/map-installation-guide). For maps that use the `serve` method, use
the [legacy installation guide](/wiki/legacy/legacy-map-installation-guide), as the process is a bit
different and differs from map to map.

- [Installation Guide](/wiki/legacy/map-installation-guide) (v1.0.0 +
  [Kronifer's Map Manager](https://github.com/Subway-Builder-Modded/subwaybuilder-patcher/releases))
- [Legacy Installation Guide](/wiki/legacy/legacy-map-installation-guide) (v1.0.0)

:::note

Any map that references "v1.0.0" will work on any newer versions of the game. Pre-v1.0.0 maps will
not work on 1.0.0 (unless you convert them following the guide) and post-v1.0.0 maps will not work
on any game version before 1.0.0. You can check what version of the game you are on by going to
settings and scrolling all the way down (it will be in the bottom-left corner).

:::
